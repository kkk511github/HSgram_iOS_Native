# HSgram App Store Metadata Draft

## App Identity

- App name: HSgram
- Bundle ID: com.hsgram.app
- Category: Social Networking

## Subtitle

安全可靠的即时通讯

## Promotional Text

HSgram 提供邮箱登录、私聊、群组、频道、媒体分享、联系人和隐私设置，继续使用现有 HSgram 服务端与 Android、PC 客户端互通。

## Description

HSgram 是面向日常沟通和团队协作的即时通讯应用。你可以使用邮箱登录，在私聊、群组和频道中收发消息、图片、视频、文件、语音消息和链接内容，并通过联系人、隐私、安全、通知和存储设置管理自己的账号体验。

本 iOS 版本是 HSgram 原生重构客户端，使用 HSgram 自有应用身份、图标、权限说明和隐私清单，连接现有 HSgram 服务端协议，与 Android 和 PC 客户端保持同一账号、聊天和消息状态。

## Keywords

HSgram, chat, messenger, social, groups, channels, secure messaging

## Review Notes

Use the provided HSgram review account to sign in with email verification. Voice calls, video chat, group voice/video chat, and live streaming are intentionally deferred in this rebuild and should not appear as active features.

## Privacy Summary

- Email address, display name, user ID, device token, chat content, and selected media are used for app functionality.
- Camera, microphone, photo library, contacts, and Face ID permissions are requested only after the user chooses a feature that needs them.
- No tracking domains are declared.
